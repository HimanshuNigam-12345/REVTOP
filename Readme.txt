Configuration Instructions for revtop_demo.zip:

1. Extract Files:
Extract the contents of the "revtop_demo.zip" file to your desired location.

2. Database Setup:
Access your preferred MySQL database management tool (e.g., phpMyAdmin).
Import the SQL script provided in the "sql" folder to create the revdb database and its tables.

3. Web Server Setup:
Install XAMPP or any other web server solution of your choice.
Place the extracted "src" folder in the web server's document root directory (e.g., htdocs for XAMPP).

4. Configuration:
Open the "config.php" file inside the "src" folder.
Update the database connection details if necessary (e.g., database name, username, password).

5. Accessing the Application:
Start your web server.
Open your web browser and navigate to the localhost URL corresponding to the "src" folder (e.g., http://localhost/revtop_demo/src).
You should now be able to access and interact with the application.

6. Testing:
Test the registration, login, and question generation functionalities to ensure everything is working as expected.
